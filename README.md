# Portfolio Tatiana Baldassarre

Portfolio profesional creado con React, Vite y CSS puro para presentar mi perfil como desarrolladora Frontend junior, mis proyectos reales y ejemplos conceptuales orientados a pequeños negocios.

El objetivo de este proyecto es mostrar una web cuidada, responsive y visualmente editorial, combinando sensibilidad por el diseño con una base tecnica clara en desarrollo frontend.

## Vista general

Este portfolio esta pensado para:

- Presentar mi trabajo como desarrolladora Frontend junior.
- Mostrar proyectos reales con capturas, tecnologias y enlaces.
- Comunicar servicios web para profesionales y pequenos negocios.
- Reunir mis datos de contacto en una experiencia clara y accesible.

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- HTML5
- CSS3
- CSS Grid
- Flexbox
- Diseno responsive

Tambien incluyo en mi perfil conocimientos de backend con Spring Boot y MySQL.

## Secciones principales

- **Inicio:** hero visual con mensaje principal y llamadas a la accion.
- **Servicios:** soluciones web para negocios, landing pages, portfolios, blogs y mantenimiento.
- **Proyectos:** casos de estudio con capturas destacadas y enlaces a proyectos reales.
- **Demos:** ejemplos conceptuales para negocios locales.
- **Sobre mi:** presentacion personal y profesional.
- **Contacto:** email, WhatsApp, LinkedIn y GitHub.

## Proyectos incluidos

- **Sparkling Tech**
  Sitio web profesional para presentar servicios tecnologicos.

- **Libria**
  Aplicacion web para organizar lecturas y gestionar contenido personal.

- **La Peonia**
  Floristeria virtual con experiencia visual cuidada.

- **Portfolio artistico GBaldassarre**
  Portfolio visual para mostrar obra artistica.

## Caracteristicas

- Diseno responsive para movil, tablet, portatil y escritorio.
- Componentes React separados y reutilizables.
- Datos de proyectos y demos centralizados.
- Estilos creados con CSS puro, sin frameworks externos.
- Navegacion sticky con estado activo al hacer scroll.
- Lightbox para visualizar capturas de proyectos y demos.
- Estetica editorial, elegante y profesional.

## Instalacion y ejecucion

Clona el repositorio:

```bash
git clone https://github.com/tbaldassarrer/portfolioTatiana.git
```

Entra en el proyecto:

```bash
cd portfolioTatiana
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el entorno de desarrollo:

```bash
npm run dev
```

Genera la version de produccion:

```bash
npm run build
```
